# EvanDurfee Tap

## How do I install these formulae?

`brew install evandurfee/tap/<formula>`

Or `brew tap evandurfee/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "evandurfee/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
